# VibesAI (Prism Ecosystem)

VibesAI is a streamlined, glassmorphic web interface focused on the core creator flow:

- Centered prompt composer on Home.
- Vertical dock navigation for fast page switching.
- Advanced controls hidden behind explicit "Show Advanced" interactions.
- In-app notifications and project library foundations.
- Product naming policy: default `VibesAI`, full legal/ecosystem `VibesAI by Prism` where appropriate.

## Tech Stack

- React + TypeScript + Vite
- React Router

## Run Locally

1. Install Node.js 20+ from the official website.
2. Install dependencies:

```bash
npm install
```

3. Start dev server:

```bash
npm run dev
```

4. Build production bundle:

```bash
npm run build
```
